# Project Constitution

This repository is the template foundation for a GitHub-native AI engineering workflow.

## Current scope

At this stage, this repository contains the initial governance and documentation baseline.

Later phases will add:

- Claude Code skills
- Claude Code subagents
- CI and automation
- issue and PR templates
- review and maintainability gates

## Hosting baseline

The default target environment for this repository is:

- GitHub Organization
- GitHub Free
- Public repository

Do not assume private-repository-only governance features.

## Working principles

1. Prefer small, reviewable increments.
2. Keep repository conventions explicit.
3. Document any setup that cannot live directly in git.
4. Keep all future automation aligned with sprint-based delivery.
5. Treat this repository as a reusable template, not as an application repository.

## Language policy

Default language for agent-facing files is English.

## Forbidden actions

- Do not add project-specific business logic here.
- Do not store secrets in the repository.
- Do not bypass documented setup steps.
