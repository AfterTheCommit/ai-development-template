# GitHub Label Setup

## Purpose

This document records the label set required by the template repository.

## Scope

Labels are repository-specific and must be created in each repository.

## Required labels

### Type
- type:feature
- type:bug
- type:spike
- type:task

### Area
- area:backend
- area:frontend
- area:database
- area:docs
- area:infra
- area:mixed

### State
- state:blocked
- state:needs-review
- state:needs-clarification

### AI workflow
- ai:claude
- ai:human-review-required

### Priority
- priority:p0
- priority:p1
- priority:p2
- priority:p3

## Rule

Use labels for quick filtering.

Use project fields for structured planning and reporting.
