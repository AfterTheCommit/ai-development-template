# Contributing

This repository is maintained as a reusable template.

## Rules

- Keep agent-facing files in English by default.
- Keep documentation precise and operational.
- Every change must preserve template reusability.
- Any GitHub UI configuration must be documented under `docs/setup/`.

## Commit style

Use clear conventional-style commit messages, for example:

- docs: add platform decision baseline
- chore: create template repository skeleton
- docs: document manual GitHub setup steps
